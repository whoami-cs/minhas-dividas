# Product Overview

## Purpose
Gerenciador de Dívidas e Empréstimos (Debt and Loan Manager) is a comprehensive financial management application designed to help users track, manage, and strategize their debt repayment. The system provides real-time tracking of credit card debts and personal loans with detailed analytics and AI-powered financial assistance.

## Value Proposition
- **Centralized Debt Management**: Single platform to monitor all credit card debts and loans
- **Financial Intelligence**: AI-powered chat assistant for personalized financial advice
- **Data Persistence**: Cloud-based storage using Supabase for reliable data access
- **Detailed Analytics**: Track interest accumulation, payment history, and debt evolution
- **Negotiation Tracking**: Manage and track debt negotiation offers
- **Goal Setting**: Create and monitor savings goals alongside debt management

## Key Features

### Credit Card Debt Management
- Track multiple credit card debts with original and current values
- Monitor interest accumulation and growth percentages
- View next month estimates and payment projections
- Attach documents and receipts to debt records
- Track negotiation offers and special payment conditions

### Loan Management
- Comprehensive loan tracking with installment details
- Payment history with amortization schedules
- Balance evolution tracking over time
- Support for early payments and discounts
- Late fee and interest calculations
- Document attachment for loan agreements

### Financial Dashboard
- Overview of total debts and financial health
- Visual charts and graphs for debt analysis
- Income tracking and budget management
- Savings goals with progress monitoring
- Monthly financial summaries

### AI Financial Assistant
- Conversational AI powered by Google Gemini
- Personalized financial advice based on user data
- Debt repayment strategy recommendations
- Budget optimization suggestions
- Context-aware responses using actual financial data

### Document Management
- Secure file upload and storage
- PDF parsing for automatic data extraction
- Attachment linking to debts and loans
- Cloud storage via Supabase buckets

## Target Users
- Individuals managing multiple credit card debts
- People with active personal loans seeking better tracking
- Users looking to optimize debt repayment strategies
- Anyone seeking financial organization and planning tools
- Brazilian users (primary language: Portuguese)

## Use Cases
1. **Debt Consolidation Planning**: Analyze all debts to create repayment strategies
2. **Interest Monitoring**: Track how interest accumulates over time
3. **Payment Scheduling**: Plan and track installment payments
4. **Negotiation Management**: Record and follow up on debt settlement offers
5. **Financial Goal Setting**: Balance debt repayment with savings objectives
6. **Document Organization**: Keep all financial documents in one secure place
7. **AI Consultation**: Get personalized advice on financial decisions
